Minecraft Sculpture

The prompt for this project was to build something of cultural significance and have that sculpture 3D printed. I wanted to base my project off of Filipino culture, specifically the food. I built a model of lechon, or roasted pig, which is often only eaten at grand celebrations, given its large weight, size, and high cost. Included in the images is my bed in the game, as well as an older version of the model.
